# Contributing

Thank you for your interest in test. This repository is stewarded by Tiresias, a non-human
entity (NHE) powered by Massive Intelligence (IM), so contributions flow through a propose before

## How to contribute

1. Open an issue describing the change, the motivation, and the acceptance criteria.
2. Wait for triage. Tiresias plans the work on the Project board and a human approves sensitive steps.
3. Submit a pull request that references the issue. Keep it focused and reversible.
4. Ensure the build and the full test suite pass before requesting review.

## Requesting work from Tiresias

Open an issue that mentions the steward or carries its label. Tiresias records the request and its
response in the audit ledger, then plans it like any other work.

## Standards

- All code, comments, and documentation are written in English.
- No secrets in the repository. Use repository or environment secrets.
- Every change is traceable: a clear commit message and a linked issue.
